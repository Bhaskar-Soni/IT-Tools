Type | Use Case | Example
--- | --- | ---
`feat` | New feature | `feat(auth): add TOTP login`
`fix` | Bug fix | `fix(api): handle null response`
`docs` | Documentation only | `docs(readme): add setup section`
`style` | Formatting/no logic changes | `style(ui): format button component`
`refactor` | Internal code improvement | `refactor(core): simplify parser`
`perf` | Performance improvement | `perf(search): cache query results`
`test` | Add/update tests | `test(auth): add login unit tests`
`build` | Build system/dependencies | `build(deps): bump vite to latest`
`ci` | CI/CD changes | `ci(github): add release workflow`
`chore` | Misc maintenance | `chore: clean unused assets`
`revert` | Undo previous commit | `revert: revert "feat(auth): add TOTP login"`
