# Linux Terminal Basics

### Open terminal and check shell
```
echo $SHELL
pwd
whoami
```

### Basic commands you should know
```
ls
ls -la
clear
history
```

### Get help for commands
```
man ls
ls --help
```

### Print text and variables
```
echo "Hello Linux"
echo $HOME
```

### Create and remove directories
```
mkdir demo-folder
rmdir demo-folder
```

### Exit terminal session
```
exit
```

### Quick reference
- Use `man <command>` when stuck.
- Use `history` to repeat previous commands.
